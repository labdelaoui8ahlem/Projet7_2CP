FAQ

Which architectural tactics are detected?

Right now, the tool supports detection of several common tactics such as PingEcho, Maintain Multiple Copies, and Id Password. We plan to expand this list as the tool evolves.

What kind of files can I upload?

You can upload plain text files that contain execution traces. Just make sure the file has a correct structure as provided in the uploading page so the parser can do its job effectively.

What happens to my files after upload?

Your file is only used temporarily during the analysis and is not stored afterward. Your data stays private and secure.

Can I export the results?

Absolutely! Once your trace is analyzed, you can export the results either as a JSON file , TXT file or a PDF file to use in reports, documentation, or further analysis.

Do I need to install anything?

Not at all, our tool runs entirely in your browser. Just upload your trace file and get a detailed analysis on your implemented architectural tactics automatically!
